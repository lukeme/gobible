Developer

The developer section contains information for creating new Go Bible translations.

For translation related questions please see the Go Bible Google Group.

The Go Bible source code is available from the Go Bible Google Code project page.

Click on a link below for more info:

  • GoBibleCreator - GoBibleCreator is a Java program that takes a ThML or OSIS file and produces Go Bible JAR and JAD files that can be loaded onto a mobile phone. This page allows you to download GoBibleCreator along with instructions on how to use it.
  • OSIS - Contains information about the OSIS format required by GoBibleCreator.
  • ThML - Contains information about the ThML format required by GoBibleCreator.
